# StockData

> StockData.org provides real-time, intraday, and historical stock, forex, and cryptocurrency market data along with global financial news and sentiment analysis. The REST API covers US-listed stocks (quotes, OHLCV, splits, dividends) and entity-level news sentiment from 5,000+ sources in 30+ languages, tracking 150,000+ entities across 70+ markets. All endpoints are GET requests authenticated with an api_token query parameter from the free dashboard.

Generated by API Evangelist from the machine-readable artifacts in https://github.com/api-evangelist/stockdata (no provider-published llms.txt exists; /llms.txt returned 404 on 2026-07-22).

## APIs

- [StockData Stock Data API](https://www.stockdata.org/documentation#stock-data): Real-time quotes, adjusted/unadjusted intraday OHLCV, end-of-day history (stocks, crypto, forex), splits, and dividends at https://api.stockdata.org/v1
- [StockData News API](https://www.stockdata.org/documentation#news-feeds): Global financial news with entity-level sentiment analysis, similar-news lookup, intraday/aggregated news statistics, trending entities, and sources
- [StockData Entities API](https://www.stockdata.org/documentation#stock-metadata): Entity search plus entity-type and industry metadata for 150,000+ tracked entities

## Conventions

- Authentication: api_token query parameter on every request
- Pagination: page + limit query parameters; responses carry meta {found, returned, limit, page}; max result set 20,000
- Rate limiting: 60-second sliding window signaled via X-RateLimit-Limit; daily plan quota signaled via X-UsageLimit-Limit; 429 rate_limit_reached, 402 usage_limit_reached
- Errors: {"error": {"code", "message"}} envelope
